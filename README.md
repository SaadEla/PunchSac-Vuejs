# PunchSac-Vuejs

## Notions dealed in the project

* Dynamic CSS classes:

```shell
      Allows us to link an html file tag dynamically.
      v-bind:style
      v-bind:class
```

* V-ON:

```shell
      We use the v-on directive to attach event listeners that invoke methods on our View instances
      v-on:click
```
